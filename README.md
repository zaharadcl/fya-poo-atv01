# fya-poo-atv01
Este repositório foi desenvolvido como parte de uma atividade prática de Programação Orientada a Objetos (POO), com o objetivo de aplicar conceitos fundamentais da área através da implementação de classes em C#.

Foram trabalhados conceitos como:

Encapsulamento
Construtores
Propriedades (get/set)
Campos somente leitura
Campos calculados
Sobrescrita de métodos (ToString)
Validações e regras de negócio

📁 Estrutura do Repositório

O projeto está organizado em pastas separadas para cada exercício:

afya-poo-atv01/
│
├── Ex01_Lampada/
├── Ex02_Cofre/
├── Ex03_Conta/
├── Ex04_RPG/

Cada pasta contém sua própria implementação e arquivos necessários para execução.

📌 Observações:
Este projeto tem fins acadêmicos e foi desenvolvido para prática dos conceitos de Programação Orientada a Objetos.
